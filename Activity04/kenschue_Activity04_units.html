The effect of 'font-size: 5vh' on the div is that the first header scales 5x with the verticle height of the webpage while 'font-size: 3vw' scales 3x with the webpages horizontal height.
'font-size: 200%' allows the font to scale with the size of webpage, this allows the type to fill the disired amount on the webpage.
Using both of these techiniques a web developer is able to create a webpage that can be viewed and legiable on a multitude of devices and screen sizes. Also, can allow for the disred design of the webpage to stay constistant across devices.
